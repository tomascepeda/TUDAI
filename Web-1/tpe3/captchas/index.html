Web Design

Con un excelente diseño Web se aumentan las posibilidades de que haya una mejor conexión con cada persona que visita nuestra página, que lea el contenido y cumpla con acciones más específicas como suscribirse, dejar comentarios, recomendar la página, etc.

Antes de enfocarse en cómo va a ser tu diseño Web, es muy importante definir tu objetivo principal. Todos los elementos que vamos a ver en este tutorial van a ser una ayuda y soporte para que el objetivo se pueda cumplir.

Layouts Designs

La noción de layout suele utilizarse para nombrar al esquema de distribución de los elementos dentro un diseño. Es habitual que un diseñador que se dedica a la creación de páginas web desarrolle un layout y se lo presente a su cliente para que éste lo apruebe y decida sobre la distribución de los contenidos.

Box Model

Todos los elementos HTML se pueden considerar como cuadros. En CSS, el término "modelo de caja" se usa cuando se habla de diseño y diseño. El modelo de cuadro CSS es esencialmente un cuadro que envuelve todos los elementos HTML. Consiste en: márgenes, bordes, relleno y el contenido real.
